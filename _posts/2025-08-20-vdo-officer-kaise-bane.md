## VDO (ग्राम विकास अधिकारी) क्या होता है?
VDO यानी **ग्राम विकास अधिकारी (Village Development Officer)**, ग्रामीण विकास मंत्रालय के अंतर्गत एक सरकारी पद है। जैसा कि नाम से ही स्पष्ट है, VDO किसी भी गांव के विकास कार्यों के लिए जिम्मेदार मुख्य प्रशासनिक अधिकारी होता है।

सरल शब्दों में, VDO सरकार और गांव की जनता के बीच एक पुल की तरह काम करता है। सरकार गांव के लोगों का जीवन बेहतर बनाने के लिए जो भी योजनाएं (जैसे स्कूल, अस्पताल, सड़क, पानी, बिजली) चलाती है, उन्हें जमीन पर सफलतापूर्वक लागू करवाने की जिम्मेदारी VDO की ही होती है।

## VDO परीक्षा प्रक्रिया और पैटर्न
VDO की भर्ती हर राज्य का अपना अधीनस्थ सेवा चयन आयोग (Subordinate Services Selection Commission) करता है।
* **फॉर्म कब आते हैं?** भर्ती का कोई निश्चित समय नहीं होता। जब भी सरकार को कर्मचारियों की आवश्यकता होती है, वैकेंसी निकाली जाती है। इसलिए आपको अपने राज्य की आधिकारिक वेबसाइट पर नजर रखनी चाहिए।

## VDO की तैयारी कैसे करें?
1.  **सिलेबस को समझें:** सबसे पहले सिलेबस को अच्छी तरह से देखें और हर टॉपिक को समझें।
2.  **टाइम-टेबल बनाएं:** हर विषय को बराबर समय दें। जिस विषय में आप कमजोर हैं, उसे हर दिन थोड़ा अतिरिक्त समय दें।
3.  **हिंदी को प्राथमिकता दें:** हिंदी एक स्कोरिंग विषय है। इसमें अच्छी पकड़ बनाकर आप मेरिट में दूसरों से आगे निकल सकते हैं।
4.  **पिछले वर्षों के प्रश्नपत्र हल करें:** इससे आपको परीक्षा के पैटर्न और प्रश्नों के स्तर का सही अंदाजा हो जाएगा।
5.  **मॉक टेस्ट दें:** नियमित रूप से मॉक टेस्ट देने से आपकी स्पीड बढ़ेगी और आप नेगेटिव मार्किंग से बचना सीखेंगे।

document.addEventListener('DOMContentLoaded', () => {
    // Logic for ALL Accordions
    const accordions = document.querySelectorAll('.accordion-container');
    accordions.forEach(accordion => {
        const headers = accordion.querySelectorAll('.accordion-header');
        headers.forEach(header => {
            header.addEventListener('click', () => {
                const content = header.nextElementSibling;
                const isActive = header.classList.contains('active');
                
                // This part makes sure only one item in THIS accordion is open at a time
                const allHeadersInThisAccordion = accordion.querySelectorAll('.accordion-header');
                allHeadersInThisAccordion.forEach(h => {
                    if (h !== header) {
                        h.classList.remove('active');
                        h.nextElementSibling.style.maxHeight = '0';
                    }
                });

                if (!isActive) {
                    header.classList.add('active');
                    content.style.maxHeight = content.scrollHeight + 'px';
                } else {
                    header.classList.remove('active');
                    content.style.maxHeight = '0';
                }
            });
        });
    });
});
</script>
